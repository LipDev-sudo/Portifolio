# LipDev.BR - Portfolio

Portfolio profissional de Hamilton Felipe Soares da Silva. Desenvolvedor Frontend especializado em React, TypeScript e Next.js.

## Preview

[Ver Demo](https://lipdev-portfolio.vercel.app/)

## Tecnologias

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS
- Framer Motion
- EmailJS
- Zod + React Hook Form

## Secoes

- **Hero** - Apresentacao com cards de stats animados
- **Sobre** - Quem sou e meus diferenciais
- **Habilidades** - Stack tecnica com cards coloridos
- **Servicos** - 6 servicos com precos
- **Projetos** - Portfolio de projetos com links e demos
- **Contato** - Formulario com EmailJS + WhatsApp

## Como rodar

```bash
npm install
cp .env.example .env.local  # Configure suas variaveis
npm run dev
```

## Deploy

Deploy automatico na Vercel a cada push na branch `main`.
